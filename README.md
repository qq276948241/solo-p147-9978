# 终端卡牌爬塔 (Slay the Spire-like Terminal Game)

基于 Python curses 的卡牌爬塔小游戏。

## 游戏特性

- 🗺 **5层地图**：每层4-6个节点，包含战斗、休息、宝箱、商店
- 🎴 **回合制卡牌战斗**：每回合抽5张手牌，3点能量
- ⚔ **卡牌类型**：攻击(6-15伤害)、防御、回血
- 👹 **敌人意图**：头顶显示下一步动作（如"攻击12"）
- 🏆 **卡牌奖励**：胜利后从3张候选卡选1加入永久卡组
- 👑 **BOSS战**：第5层最终BOSS
- 📺 **三栏布局**：左玩家状态、中战场、底日志
- ✨ **状态效果系统**：中毒、虚弱、燃烧、力量、荆棘等多种Buff/Debuff

## 状态效果说明

| 状态 | 图标 | 类型 | 效果 |
|---|---|---|---|
| 中毒 | ☠ | Debuff | 每回合受到固定伤害 |
| 虚弱 | 💫 | Debuff | 攻击力减半 |
| 燃烧 | 🔥 | Debuff | 每回合受到伤害，且伤害每回合+3 |
| 力量 | 💪 | Buff | 每层增加50%攻击力 |
| 荆棘 | 🌵 | Buff | 受到攻击时反弹固定伤害 |

## 新卡牌列表

- **毒刃** [1费]：造成4伤害，使敌人中毒(5/3回合)
- **剧毒** [2费]：使敌人中毒(8/4回合)
- **虚弱打击** [1费]：造成5伤害，使敌人虚弱(2回合)
- **燃火术** [1费]：造成3伤害，使敌人燃烧(4/3回合)
- **烈焰爆发** [2费]：造成8伤害，使敌人燃烧(6/3回合)
- **力量药剂** [1费]：获得力量(1/3回合)
- **狂暴** [2费]：获得力量(2/4回合)
- **荆棘护甲** [2费]：获得8护甲，获得荆棘(3/3回合)

## 操作说明

### 地图界面
- `W/A` 或 `↑` - 向上选择节点
- `S/D` 或 `↓` - 向下选择节点
- `Enter` - 确认选择
- `Q` - 退出游戏

### 战斗界面
- `1-5` - 打出对应位置的手牌
- `空格` - 结束回合
- `Q` - 退出游戏

### 其他界面
- 卡牌奖励：`1-3` 选择卡牌，`S` 跳过
- 休息点：`1` 休息回血
- 宝箱：`Enter` 开启
- 商店：`1-5` 购买卡牌，`H` 恢复HP，`Q` 离开

## 运行方式

```bash
# 安装依赖（Windows）
pip install windows-curses

# 启动游戏
python main.py
```

> 注意：终端窗口需要至少 80x25 大小，建议使用支持256色的终端。
